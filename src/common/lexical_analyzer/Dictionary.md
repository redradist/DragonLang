Smallest character in Dragon dictionary is a token.
There are 7 different tokens exist:
1. SpecialSymbol:\
   < > + - = - > " ' ...
2. Keyword:\
   fun return class ...
3. ContextKeyword:\
   in, is, not, as ...
4. NewLine:\
   \r \n
5. NumberConst:\
   23, 27.23, 1, 3, ...
6. TextConst:\
   'Hello Denis Good Job', "Hello Denis Good Job" ...
7. SymbolId:\
   Name of variables, name of classes, name of structs and so on ...
